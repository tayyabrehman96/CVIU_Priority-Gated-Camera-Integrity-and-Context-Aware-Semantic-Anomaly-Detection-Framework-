# Figures (not in Git)

PNG assets for the manuscript and diagrams are **not** tracked in this repository (keeps the clone small).

- Run from the repo root: `python generate_methodology_diagram.py`, `python generate_sota_figures.py`, etc., to populate this folder or the root with outputs.
- For LaTeX builds, keep manuscript `.tex` / `.bib` locally and place the expected figure filenames here (or update paths in your copy).
